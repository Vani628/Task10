Usage of Inheritance, Built-In Functions, Mathematical Operations

Inheritance

Inheritance in SCSS allows you to reuse styles from one selector in another, following the DRY (Don't Repeat Yourself) principle. This is done using the @extend directive.

Built in Functions

SCSS provides a variety of built-in functions that help manipulate data types such as colors, numbers, strings, and lists.

Mathematical Operations

SCSS supports basic mathematical operations like addition, subtraction, multiplication, and division. You can perform these operations directly in your styles.
